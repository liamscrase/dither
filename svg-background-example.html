üìù How to Use

Step 1: Generate Your SVG

Use the Python script to generate your custom dithered gradient:

python3 generate_dither_svg.py > my-gradient.svg

Step 2: Customize (Optional)

Edit the script to change colors, size, density, etc:

COLOR_INNER = (45, 10, 78) # Dark purple COLOR_MIDDLE = (123, 31, 162) # Medium purple COLOR_OUTER = (225, 190, 231) # Light purple PIXEL_SIZE = 3 # Size of dither pixels DENSITY_CURVE = 1.5 # How fast density increases

Step 3: Use in CSS

Reference the generated SVG file:

.hero-section { background-image: url('my-gradient.svg'); background-size: cover; background-position: center; }
